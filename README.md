# AI Academic Research Assistant

## Project Overview

The AI Academic Research Assistant is a Python-based software project designed to help users analyze academic documents more easily. The system can read text files and PDF documents, extract important keywords, and generate short summaries automatically.

The project was developed using a modular structure where different tools work together inside one integrated workflow.

---

# Features

- Read text files
- Read PDF documents
- Extract important keywords
- Generate automatic summaries
- Integrated research agent workflow
- Error handling system
- Automated testing using pytest
- GitHub version control

---

# Technologies Used

- Python
- PyPDF2
- YAKE
- Sumy
- Pytest
- Git and GitHub
